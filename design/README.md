# 🎨 Mudipu Design System

This folder contains the complete Mudipu design system, brand guidelines, and templates for all frontend projects.

---

## 📁 Contents

### Documentation
- **DESIGN_FILES_README.md** - Overview of all design files
- **DESIGN_SYSTEM.md** - Complete design guidelines and component documentation
- **DESIGN_TEMPLATE_README.md** - Quick start guide for new projects
- **design-reference.html** - Visual reference guide (open in browser)

### Design Assets
- **design-tokens.json** - Exportable design tokens (colors, typography, spacing)
- **mudipu-logo.svg** - SVG logo file

### Templates
- **tailwind.config.template.js** - Tailwind CSS configuration
- **index.css.template** - Theme CSS with light/dark mode variables
- **package.json.template** - Dependencies template

---

## 🚀 Quick Start

1. **View Design Reference:** Open `design-reference.html` in your browser
2. **Read Guidelines:** Start with `DESIGN_SYSTEM.md`
3. **Start New Project:** Follow `DESIGN_TEMPLATE_README.md`
4. **Use Tokens:** Import `design-tokens.json` for programmatic access

## 🎯 Brand Identity

**Colors:**
- Primary Red: `#EF4444`
- Light Background: `#FAFAFA`
- Dark Background: `#0B0F19`

**Typography:**
- Font: JetBrains Mono
- Logo: Red dot + "mudipu" wordmark

**Components:**
- Logo, ThemeToggle, Cards, Buttons, Badges
- Located in `../src/components/`
